Sobre o Projeto

Este site foi desenvolvido por Matheus Santos Dolenga como parte da Atividade Somativa 1 da disciplina de Fundamentos de Programação Web.

O objetivo foi criar um site simples utilizando apenas HTML, CSS e JavaScript puros, sem o uso de frameworks, para demonstrar a compreensão dos conceitos fundamentais do desenvolvimento front-end.

Tecnologias Utilizadas:

  • HTML5: Para a estruturação semântica do conteúdo.
  • CSS3: Para a estilização, layout e design responsivo, com uso de classes reaproveitáveis e variáveis CSS.
  • JavaScript: Para a validação dinâmica do formulário e para a manipulação dos dados recebidos via método GET.
